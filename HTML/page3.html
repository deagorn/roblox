CSS: Стилізація веб-сторінок

CSS (Cascading Style Sheets) використовується для визначення стилю веб-сторінок, включаючи кольори, шрифти, макети тощо.

Приклад використання CSS

            body {
                font-family: Arial, sans-serif;
            }

            h1 {
                color: blue;
            }

            p {
                color: gray;
            }
